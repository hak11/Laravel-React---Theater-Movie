# Laravel React - Movie Theater

- Laravel 8
- React 16.2

## Installation


clone this repo first

```bash
composer install

npm install
```

create .env and fill it

```bash
php artisan migrate

php artisan db:seed

php artisan serv
```

if you want edit react file, run this command

```bash
npm run watch
```
